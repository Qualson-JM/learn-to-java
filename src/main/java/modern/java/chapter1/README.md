# Chapter1. 자바 8, 9, 10, 11

### 자바의 대중적인 프로그래밍으로서의 성장
- 캡슐화  
- 객체지향

### 병렬성 처리시 안전하게 실행할 수 있는 코드를 만드려면
- 가변데이터(mutable data)에 접근하지 않아아야 한다.
- 순수함수 (pure)
- 부작용 없는 함수 (side-effect free)
- 상태없는 함수 (stateless)

### 자바가 진화해야하는 이유
- 객체지향은 그 작업을 어떻게 수행하는지에 초점
- 함수형 프로그래밍은 결과값을 생성하는 데에 초점
(작업 수행은 파라미터로 던져짐)
